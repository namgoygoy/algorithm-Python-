## 1379번

https://www.acmicpc.net/problem/1379

### 문제 설명

N개의 강의 시작·종료 시간이 주어질 때, 모든 강의를 진행하기 위해 필요한 최소 강의실의 개수와 각 강의별 배정된 강의실 번호를 구하는 문제

### 로직

1. **최소 힙:** 사용 중인 강의실의 종료 시간을 관리한다. 가장 빨리 비는 강의실을 찾는다.
2. `가장 빠른 종료 시간` ≤ `현재 강의 시작 시간`이면, 해당 방을 이어서 사용한다.
3. 그렇지 않으면, 새로운 강의실을 개설한다.

### 코드

```python
import sys
# 가장 빨리 끝나는 강의실(최솟값)을 즉시 확인하기 위해
# 리스트 O(N) 대신, 우선순위 큐(힙) O(log N) 사용
import heapq

input = sys.stdin.readline

N = int(input())
lectures = []

for _ in range(N):
    lectures.append(list(map(int, input().split())))

# 강의 시작 시간 기준으로 정렬 (강의실 배정을 위해)
lectures.sort(key=lambda x: x[1])

min_heap = []
# 각 강의실에 배정된 강의 ID를 저장할 리스트
rooms = [0] * (N + 1)
cnt = 0

for id, start, end in lectures:
    # 가장 빨리 끝나는 강의실이 현재 강의 시작 시간 이전에 끝난다면 재사용  
    if min_heap and min_heap[0][0] <= start:
        _, room_id = heapq.heappop(min_heap)
        heapq.heappush(min_heap, (end, room_id))
        rooms[id] = room_id
    # 새로운 강의실이 필요한 경우    
    else:
        cnt += 1
        heapq.heappush(min_heap, (end, cnt))
        rooms[id] = cnt

print(cnt)
for i in range(1, N + 1):
    print(rooms[i])
```
