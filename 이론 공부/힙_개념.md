## 힙 자료구조

힙은 '최대값이나 최소값'을 빠르게 찾기 위해 고안된 **완전 이진 트리** 기반의 자료구조이다. **우선순위 큐**를 구현하는 데 자주 사용된다.

- `heappush()` : 힙에 원소를 추가한다. 시간복잡도는 $O(log N)$이다.
- `heappop()` : 힙에 가장 작은 값을 제거/반환한다. 시간복잡도는 $O(log N)$이다.
- `heapify()` : 리스트를 힙으로 변환하는 함수이다. 시간복잡도는 $O(N)$이다.

### Push

```python
from heapq import heappush, heappop

heap = []

heappush(heap, 4)
heappush(heap, 2)
heappush(heap, 8)
heappush(heap, 1)

print(heap)
# [1, 2, 4, 8]
```

### Pop

```python
from heapq import heappush, heappop

heap = []

heappush(heap, 4)
heappush(heap, 2)
heappush(heap, 8)
heappush(heap, 1)

print(heappop(heap))
# 1
print(heap)
# [2, 4, 8]
```

### Heapify

```python
from heapq import heapify

heap = [4, 1, 7, 3, 8, 5]
heapify(heap)
print(heap)
# [1, 3, 5, 4, 8, 7]
```

## 1927 최소 힙

https://www.acmicpc.net/problem/1927

### **문제 설명**

연산의 개수 N을 입력받는다. 입력이 자연수 `x`면 힙에 추가하고, 0이면 힙에서 가장 작은 값을 출력하고 제거한다. (비었으면 0을 출력 한다)

### 로직

1. 입력 `x`가 0보다 크면, `heapq.heappush()`를 사용해 힙에 값을 추가한다.
2. 입력 `x`가 0이면, 힙이 비어있는지 확인한다.
3. 비어있지 않으면, 최소값을 꺼내 출력하고, 비어있으면 0을 출력한다.

## 코드

```python
import heapq
import sys

input = sys.stdin.readline

heap = []

N = int(input())

for _ in range(N):
    x = int(input())

    if x > 0:
        # 최소 힙에 x를 추가
        heapq.heappush(heap, x)
        
    elif x == 0:
        if heap:
            # 힙이 비어있지 않다면, 가장 작은 값을 pop하여 출력
            print(heapq.heappop(heap))
        else:
            # 힙이 비어있다면 0을 출력
            print(0)
```
