## 정렬 알고리즘

**정렬 알고리즘**은 데이터 집합(리스트, 배열 등)의 원소들을 **특정한 기준**에 따라 **순서대로 나열**하는 방법이다.

### K번째 수

```jsx
def solution(array, commands):
    answer = []
    for i, j, k in commands:
        arr = array[i-1:j]
        arr.sort()
        answer.append(arr[k-1])
    return answer
```

### sort()함수

Python의 **`list.sort()`** 및 내장 함수 `sorted()`는 **팀소트(Timsort)**를 사용한다.

**팀소트**는 **하이브리드 안정 정렬** 알고리즘으로, **병합정렬**과 **삽입정렬**의 장점을 결합한 알고리즘이다.

- **최선의 경우:** O(N)
- **최악의 경우:** O(NlogN)
