## 9663번 **N-Queen**

**N×N 크기의 체스판 위에 N개의 퀸을 배치**하되, 어떤 퀸도 **서로 공격할 수 없도록** 배치하는 문제

N개의 퀸은 모두 **서로 다른 행, 다른 열, 그리고 서로 다른 대각선에 배치**

## 로직

1. 해당 열이나 대각선이 이미 사용 중이면 건너뛴다.
2. 유효한 자리는 True로 바꾸고 다음행을 탐색한다.
3. 다음 행을 탐색하고 돌아오면, 다시 시도할 수 있게 False로 복구한다.

## 코드

```python
n = int(input())

col_used = [False] * n
# 배열의 크기 2n-1
diag1_used = [False] * (2 * n - 1)
diag2_used = [False] * (2 * n - 1)

count = 0 

def backtrack(row):
    global count, n
    # 성공적으로 n퀸을 배치한 경우
    if row == n:
        count += 1
        return
    

    for col in range(n):
        # 해당 열과 대각선이 이미 사용된 경우 건너뜀, backtrack(row)
        if col_used[col]:
            continue
        if diag1_used[row + col]:
            continue
        if diag2_used[row - col + n - 1]:   
            continue
        
        # 해당 열과 대각선을 사용 표시
        col_used[col] = True
        diag1_used[row + col] = True
        diag2_used[row - col + n - 1] = True
        # 다음 행으로 이동
        backtrack(row + 1)
        # 사용 표시 해제 (백트래킹)
        col_used[col] = False
        diag1_used[row + col] = False
        diag2_used[row - col + n - 1] = False

backtrack(0)    

print(count)
```

## 1987 알파벳

R x C 크기의 보드에 알파벳이 적혀 있다. (0, 0) 칸에서 시작하여 상하좌우로 인접한 칸으로 이동한다.

**지금까지 지나온 모든 칸에 적힌 알파벳과 다른 알파벳이 적힌 칸으로만** 이동할 수 있다.

## 로직

1. 현재 위치에서 4방향으로 다음 위치를 탐색한다.
2. 이미 탐색한 알파벳인 경우 건너뛴다.
3. 탐색하지 않은 알파벳의 경우 방문 처리하고 `count`를 1 증가시켜 재귀 호출한다.
4. 탐색이 모두 끝나고 현재 위치로 되돌아오면 다른 경로도 탐색할 수 있도록 해당 알파벳을 다시 제거한다.

## 실패 코드 (메모리 초과)

```python
import sys

# 깊은 재귀 한도 설정
sys.setrecursionlimit(10**6)

input = sys.stdin.readline

R, C = map(int, input().split())
# 보드 입력
board = [list(input().strip()) for _ in range(R)]

# 상하좌우 이동 방향
dx = [-1, 1, 0, 0]
dy = [0, 0, -1, 1]

# 방문 기록을 위한 집합
# 리스트 대신 set을 쓰는 이유: set 안에 있는지 확인하는 속도(in 연산)이 리스트보다 훨씬 빠름
visited_alphabets = set()
max_count = 1

def dfs(x, y, count):
    global max_count
    max_count = max(max_count, count)

    # 네 방향으로 이동
    for i in range(4):
        nx = x + dx[i]
        ny = y + dy[i]

        # 보드 범위 내에 있고, 방문하지 않은 알파벳인 경우
        if 0 <= nx < R and 0 <= ny < C:
            if board[nx][ny] not in visited_alphabets:

                # 방문 처리 및 재귀 호출
                visited_alphabets.add(board[nx][ny])
                dfs(nx, ny, count + 1)

                # 백트래킹: 방문 처리 해제
                visited_alphabets.remove(board[nx][ny])

visited_alphabets.add(board[0][0])
dfs(0, 0, 1)
print(max_count)
```
